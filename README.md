# IpOsint
OSINT for IP addresses including geolocation, ping and port scanning/checking written in bash.

# How to use
- git clone https://github.com/invalidrandom/IpOsint/
- cd IpOsint && cd iposint
- chmod +x *
- ./iposint.sh

# Tool purpose
This tool has been designed to help OSINT IP addresses without any effort at all, making it easy for anyone to scan IP adresses.

# Features
- Geolocate IP/Domain
- Ping IP
- NMAP specific port search
